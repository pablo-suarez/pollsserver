# Development

1. Clone the project
2. Copy ```env.template``` and rename ```.env```
3. Run
```
npm install
```
4. Set up the image (Docker desktop)
```
docker-compose up -d
```
5. Run backend Nest
```
npm run start
```
6. Check site
```
localhost:4000
```
